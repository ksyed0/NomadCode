# Product Name

## Product: Mobile IDE

**Version:** 0.1
**Date:** 2026-03-09

---

## Working Title

**NomadCode**

---

## Naming Criteria

A strong product name should be:
- Memorable and easy to pronounce
- Relevant to the mobile/on-the-go developer use case
- Available as an App Store / Play Store listing
- Available as a domain name (or close variant)
- Trademark-safe (legal review required)

---

## Name Candidates

| Name | Pros | Cons | Score (1–10) |
|---|---|---|---|
| **NomadCode** | Evokes mobile/travel, unique, memorable | Somewhat long | 8 |
| **PocketIDE** | Descriptive, clear | Generic, possibly trademarked | 6 |
| **CodeAway** | Playful, memorable | Could imply "throw code away" | 5 |
| **DriftCode** | Evokes movement, modern | Less clear association | 6 |
| **FieldCode** | Professional, field-work association | Less catchy | 5 |

---

## Recommended Name

**NomadCode**

*Rationale:* Directly communicates the mobile, location-independent developer lifestyle. Unique enough to be memorable and searchable. Aligns with the product's core value proposition.
