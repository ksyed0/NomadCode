# Product Roadmap

## Product: Mobile IDE

**Version:** 0.1
**Date:** 2026-03-09

---

## Vision

A best-in-class mobile IDE that makes professional software development possible anywhere, on any device.

---

## Now (v0.1 – v0.5)

**Theme: Core Experience**

- [x] Project scaffolding and monorepo setup
- [ ] Syntax-highlighted code editor (JS, TS, Python, JSON, Markdown)
- [ ] File Explorer with CRUD
- [ ] Integrated sandboxed terminal
- [ ] Command palette
- [ ] Light/dark theme
- [ ] Local file system access (device storage)

---

## Next (v0.6 – v1.0)

**Theme: Power User & Quality**

- [ ] Git integration (status, diff, commit, push/pull)
- [ ] Multi-tab editor
- [ ] Extension/plugin system (WASM sandbox)
- [ ] Code completion (language server protocol via WASM)
- [ ] Search & replace across files
- [ ] Performance optimization for large files and directories
- [ ] Accessibility audit and compliance

---

## Later (v1.1+)

**Theme: Collaboration & Ecosystem**

- [ ] Cloud sync (cross-device file access)
- [ ] Real-time collaborative editing
- [ ] Remote SSH / dev container support
- [ ] Extension marketplace
- [ ] AI code assistant integration
- [ ] Custom keybindings
- [ ] Snippet library

---

## Icebox (Considering)

- Desktop (Electron) app leveraging shared codebase
- Self-hosted backend option for enterprises
- Offline AI inference for code completion

---
