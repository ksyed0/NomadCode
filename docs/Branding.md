# Branding

## Product: Mobile IDE (NomadCode)

**Version:** 0.1
**Date:** 2026-03-09

---

## Brand Identity

### Tagline Candidates
- *"Code from anywhere."*
- *"Your IDE. Untethered."*
- *"Build without boundaries."*

**Recommended:** *"Code from anywhere."* — Simple, clear, aspirational.

---

## Visual Identity

### Color Palette

| Role | Name | Hex |
|---|---|---|
| Primary | Nomad Blue | `#2563EB` |
| Secondary | Sand | `#D97706` |
| Background (light) | Off-White | `#F9FAFB` |
| Background (dark) | Deep Slate | `#0F172A` |
| Text (light) | Charcoal | `#111827` |
| Text (dark) | Cloud | `#E2E8F0` |
| Accent | Teal | `#0D9488` |
| Error | Coral | `#EF4444` |
| Success | Sage | `#22C55E` |

### Typography

| Role | Font | Weight |
|---|---|---|
| UI (headings) | Inter | 600, 700 |
| UI (body) | Inter | 400, 500 |
| Code / Editor | JetBrains Mono | 400, 700 |

### Logo Concept

- Mark: Stylized `{ }` brackets forming a compass or nomad tent shape.
- Wordmark: "NomadCode" in Inter SemiBold.
- Icon (app icon): Compact `{ }` mark on Nomad Blue background.

## Voice & Tone

| Attribute | Description |
|---|---|
| Knowledgeable | Speaks developer-to-developer. No condescension. |
| Empowering | Helps users feel capable and productive. |
| Direct | No fluff. Gets to the point. |
| Adventurous | Celebrates the freedom of coding anywhere. |

---

## Brand Don'ts

- Do not use corporate blue (`#003087`-style colors) — too enterprise.
- Do not use overly playful language in UI copy.
- Do not clip or stretch the logo.
- Do not use more than two fonts in any single screen.
