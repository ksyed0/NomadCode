# Mobile VS Code-Compatible IDE — User Stories

## Personas
- Tablet Developer: Works primarily on iPad/iPadOS
- Phone Developer: Works on Android/iPhone
- Power User: Needs AI features and advanced extensions
- Open Source Contributor: Uses free tier, contributes extensions

---

## User Stories

### Editor
- As a tablet developer, I want drag-and-drop file navigation so I can switch files efficiently.
- As a phone developer, I want adaptive panel layout so I can edit comfortably on smaller screens.

### Extensions
- As a power user, I want cloud-run extensions so I can use heavy language servers.
- As a contributor, I want JS-only extensions to run offline in the free tier.

### Git
- As a developer, I want commit, push, and pull functionality so I can manage code versions.
- As a user, I want GitHub integration to work directly in the IDE.

### AI
- As a Pro+ user, I want inline code suggestions so I can write code faster.
- As a Pro+ user, I want AI refactoring recommendations to improve code quality.

### UX
- As a user, I want touch gestures and virtual keyboard support for smooth editing.
- As a tablet developer, I want a workspace layout similar to desktop VS Code.

### Monetization
- As a free tier user, I want ads displayed unobtrusively.
- As a Pro/Pro+ user, I want subscription-based unlocking of advanced features.
