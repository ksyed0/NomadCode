# NomadCode IDE - Prototype

Mobile-first VS Code-compatible IDE for iOS, iPadOS, and Android.

## Features
- Monaco editor in WebView
- JS-only extensions sandbox
- Local file system + GitHub
- Tablet-first responsive layout

## Setup
1. `npm install`
2. `npx react-native run-ios` or `npx react-native run-android`
